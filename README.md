# AI Financial Assistant

An AI-powered financial assistant for underserved communities, accessible directly through WhatsApp. The assistant helps users discover government schemes, detect financial scams, improve financial literacy, and receive personalized financial guidance through natural language conversations.

## Features

* Government Scheme Recommendations
* Scam Detection & Fraud Awareness
* Financial Education
* Personalized User Memory
* WhatsApp-Based Conversational Assistant

## Tech Stack

* Python
* FastAPI
* LangGraph
* LangChain
* MongoDB
* Node.js
* Baileys (WhatsApp Integration)
* MyScheme API
* OpenRouter

## Environment Variables

Create a `.env` file in the project root:

```env
OPENROUTER_API_KEY=your_openrouter_api_key
MY_SCHEME_API=your_myscheme_api_key
MONGODB_URI=your_mongodb_uri
```

## Installation

```bash
pip install -r requirements.txt
npm install
```

## Run the Application

### Backend API

```bash
py -m uvicorn api:app --reload --port 8000
```

### WhatsApp Service

```bash
node index.js
```

Scan the QR code generated by Baileys to connect a WhatsApp account.
